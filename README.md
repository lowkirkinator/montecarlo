### 🛠️ Hur man uppdaterar (för mig och andra)
Vill du lägga till något eller rätta ett stavfel?
1. Öppna `index.html` (eller relevant kursfil).
2. Klicka på penn-ikonen (Edit) uppe till höger.
3. Gör dina ändringar i HTML-koden.
4. Scrolla ner och klicka på **"Commit changes"**.
5. Hemsidan uppdateras automatiskt inom någon minut!
